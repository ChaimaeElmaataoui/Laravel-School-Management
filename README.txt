## Laravel School Management System (LSMS)

**LSM System** is a simple role based school management system.

### Installation
01. `cd laravel-school-management-system`
02. `composer install`
03. `npm install`
04. `php artisan key:generate`
05. `php artisan migrate`
06. `php artisan db:seed`
07. `php artisan serve`
